# SplitSmart

SplitSmart is a React + Vite web application to manage groups, members, and expenses. Users can create groups, add members, track expenses, and see balances. Groups can also be permanently deleted with a confirmation popup.

## Features

- Create, view, and delete groups
- Add members to groups
- Track group expenses
- View balance summary per group
- Responsive and user-friendly design

## Tech Stack

- React
- Vite
- Tailwind CSS
- LocalStorage for data persistence
